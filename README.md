# Sachin R B - Film Direction Portfolio

A cinematic, high-impact digital portfolio showcasing the creative journey and technical expertise of Sachin R B, a Film Direction student at LV Prasad College of Media Studies.

## 🎬 Project Concept

This portfolio is designed to mirror the aesthetic and emotional weight of cinema. It combines modern web technologies with a "film diary" aesthetic, featuring glassmorphism, dynamic animations, and a rich, tailored color palette.

## 🚀 Key Features

- **Cinematic Background**: A custom-built background that sets the mood for a film-focused experience.
- **Work Credits (Experience)**: A detailed showcase of projects (like the *Dialogue Project*) with nested modals for deep dives into roles, years, and screengraphs.
- **Software Skills Meter**: A visual representation of proficiency in industry-standard tools like DaVinci Resolve, Adobe Premiere Pro, Pro Tools, and Canva.
- **Equipment & Gear**: A specialized section highlighting technical familiarity with professional camera and sound equipment.
- **Influence Board (Favourites)**: A curated space for favourite films, books, and anime that shape Sachin's creative vision.
- **Responsive & Interactive**: Full mobile responsiveness with scroll-reveal animations for a polished user journey.

## 🛠 Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: Scroll-reveal logic for dynamic content entry.

## ⚡ Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run in development mode**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
